## 2. 🫧 评价

TNotes.algorithms 是一个 TNotes 知识库体系中专门用于记录和“数据结构”、“算法”相关内容的知识库。

记录的内容主要会包括：

- 学习资源及相关工具
  - 数据结构和算法的一些学习资料
  - 刷题平台
- 数据结构
- 算法
